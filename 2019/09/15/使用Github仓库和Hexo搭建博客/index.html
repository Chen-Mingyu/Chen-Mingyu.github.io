使用Github仓库和Hexo搭建博客

发表于 2019-09-15 分类于 搭建博客 ， Github+Hexo 阅读次数：
本文字数： 1.9k 阅读时长 ≈ 3 分钟

前言

许久之前一直有搭建个人博客的想法，直到最近才开始行动。
搭建博客大致流程：

  1. 注册Github账号，申请Github仓库，用于存放博客文件及提供访问地址
  2. 下载Hexo，快速生成博客框架，下载主题，美化博客
  3. 使用ssh生成访问Github的密钥，这样便于借助GitBash一键上传本地保存的博客到Github

电脑环境

Windows10 1803,Node.js 10.16.3,npm 6.9.0,Git 2.23.0

具体步骤

  1. 先安装Node.js，官网https://nodejs.org/en/下载32位/64位最新版即可(对应Windows系统位数)。官网https://gitforwindows.org/下载安装Git，与前面位数相同。在“命令提示符”中输入“node -v”，“npm -v”，“git –version”分别查看版本信息，若都有版本信息，则安装成功
  2. 如果没有Github账号，先注册，再申请仓库,注意申请仓库时，仓库名必须是：“你的Github名称.github.io.git”，否则仓库页面会“404 Not Find”
  3. 进入仓库，即“https://github.com/xxx/xxx.github.io”(xxx为你的Github账号)点击“Settings”，在“Options”里确认下仓库名是否正确，在“Github Pages”的“Choose a theme”选择一个主题，这时就可以看到提示“Your site is published at https://xxx.github.io/”，就可以进入该网站了
  4. 创建博客文件夹，打开命令提示符，进入该文件夹下，执行“npm install hexo-cli -g”，命令行执行“hexo -v”验证成功，再执行“hexo init”初始化，提示“Start blogging with Hexo!”表明安装成功，再执行“npm install hexo –save”，hexo环境就搭建好了
  5. 在博客文件夹目录下，执行“hexo init”，之后就可以使用“hexo n 博文标题”新建博文，对“source_post\xxx.md”进行编辑，编辑完成后使用“hexo g”将md文件生成html文件以及生成目录，“hexo s”开启本地预览，通过“http://localhost:4000/”即可本地预览
  6. 配置ssh，Windows10系统“应用和功能”的“管理可选功能”的“添加功能”中添加“OpenSSSH 客户端”，打开后执行“ssh-keygen -t rsa -C “邮件地址””，完成后打开电脑用户目录，找到“.ssh”文件夹，打开“.ssh\id_rsa.pub”，记事本打开并复制里面的内容，打开你的github主页，进入个人设置->SSH and GPG keys->New SSH key，粘贴到这里。打开“Git Bash”，执行“ssh -T git@github.com”，提示“Hi liuxianan! You’ve successfully xxx, but GitHub does not provide shell access.”表示成功，再执行“$ git config –global user.name “你的github用户名””，“git config –global user.email “github注册邮箱””就ok了
  7. 设置好ssh密钥后，打开博客目录下“_config.yml”，进行如下修改，注意：xxx为Github用户名
    deploy:
    type: git
    repository: git@github.com:xxx/xxx.github.io.git
    branch: master
  8. 打开“Git Bash”，转到博客目录下，执行“npm install hexo-deployer-git”，就可以在Git Bash使用“hexo d”上传本地博文到GitHub仓库了，这时就可以打开“https://xxx.github.io/”(xxx为GitHub用户名)查看新搭建好的博客了。

美化博客

  1. 改变主题，执行“git clone https://github.com/theme-next/hexo-theme-next themes/next”，如果速度慢导致失败的话，可以参考
